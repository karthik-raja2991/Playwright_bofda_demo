Playwright Commands:
===========================

To run all spec files in tests folder, 
by default in all 3 browsers (chromium,firefox,webkit) in headless
	==> npx playwright test 

To open the html reports of last execution
	==> npx playwright show-report

To run specified spec file in tests folder, 
by default in all 3 browsers (chromium,firefox,webkit) in headless
	==> npx playwright test	tests/Playwright01.spec.ts 
	

To run specified spec file in tests folder, 
by default in all 3 browsers in headed mode
	==> npx playwright test	tests/Playwright01.spec.ts --headed 
	
	
To run specified spec file in tests folder, 
in the specified browser in headed mode 
	==> npx playwright test	tests/Playwright01.spec.ts --headed --project=chromium 
	
To run specified spec file in tests folder in interactive UI mode 
	==> npx playwright test	tests/Playwright01.spec.ts --ui
	

To run specified spec file in tests folder parallelly 
	==> npx playwright test	tests/Playwright01.spec.ts --workers=3 --headed
	

To run specified spec file in debug mode (with Playwright Inspector window for stepping through the code)
	==> npx playwright test	tests/Playwright01.spec.ts --debug 
	
	
To run specified spec file along with storing trace log 
	==> npx playwright test	tests/Playwright01.spec.ts --trace on 
	==> To View Trace Log:
		==> npx playwright show-report
		==> In html report, click on trace log link
	
	
To record script using codegen:
	==> npx playwright codegen 
	==> npx playwright codegen http://blazedemo.com 
	
	
To open the trace log after execution:
	==> npx playwright show-trace <path_of_trace.zip_file>
		e.g.: npx playwright show-trace trace/trace.zip	

	
To generate different types of reports from command-line
(report types: dot,line,list,html,github,json,junit)
	==> npx playwright test tests/Playwright01.spec.ts --reporter="dot"
	==> npx playwright test tests/Playwright01.spec.ts --reporter="line"
	==> npx playwright test tests/Playwright01.spec.ts --reporter="list"
	==> npx playwright test tests/Playwright01.spec.ts --reporter="html"
	==> npx playwright test tests/Playwright01.spec.ts --reporter="github"
	==> npx playwright test tests/Playwright01.spec.ts --reporter="json"
	==> npx playwright test tests/Playwright01.spec.ts --reporter="junit"

	Best for CI/CD Pipeline execution:
		==> npx playwright test tests/Playwright01.spec.ts --reporter="list,html"

		
To set environment variable before running playwright test
	For Windows:
		SET APIKEY=ABCD1234 & npx playwright test tests/Playwright20.spec.ts 
	
	For Linux & Mac:
		APIKEY=ABCD1234 npx playwright test tests/Playwright20.spec.ts 
	
	

To Set some important timeouts:
	1. To increate max time for a test() block within which execution should complete
		==> Default: 30000 ms
		==> To change: open playwright.config.ts and add this in the beginning
			timeout: 60000

	2. To increase the navigation timeouts:
		==> Default: 0 i.e. no timeout 
		==> To change: open playwright.config.ts and add this inside use{} block
			navigationTimeout: 60000,
			
	3. To increase the timeout / auto-wait:
		==> Default: 5000 ms
		==> To change: open playwright.config.ts and add this at the top above or below reporter / retires
			  expect:{
				timeout: 10000
			  },
