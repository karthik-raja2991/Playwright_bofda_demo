Playwright Project Config Steps:
=====================================
1. Create Project Folder
	==> e.g. C:\Work\Playwright
	
2. Open VisualCode and open above folder

3. Open Terminal:
	==> npm init -y 
		- This will create package.json
		
4. Playwright Framework Setup
	4.1 Run the following command in terminal
		==> npm init playwright@latest
